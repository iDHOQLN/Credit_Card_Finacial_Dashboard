# Credit Card Financial Report — Power BI

## 📊 Overview
This project is an end-to-end Power BI dashboard built to analyze credit card 
customer and transaction data. It provides financial stakeholders with insights 
into revenue trends, customer demographics, delinquency rates, and transaction 
behavior, enabling data-driven decision-making for risk and marketing teams.

## 🎯 Objectives
- Track weekly/monthly revenue and transaction volume
- Analyze customer segmentation by income, age, and education
- Monitor delinquency rate and credit limit utilization
- Provide drill-down capability from summary to customer-level detail

## 🛠️ Tools & Technologies
- Power BI Desktop
- Power Query (ETL / data cleaning)
- DAX (calculated measures & KPIs)
- Data Modeling (star schema)

## 📁 Project Structure
- `data/` — raw and cleaned datasets (CSV)
- `pbix/` — Power BI report file
- `images/` — dashboard screenshots
- `docs/` — data dictionary & DAX measure documentation

## 📈 Key Insights
- Revenue trend analysis by week/quarter
- Customer segmentation (income group, age bracket, card category)
- Delinquency and risk indicators
- Activation rate and customer acquisition trend

## 🚀 How to Use
1. Clone the repo
2. Open `pbix/Credit_Card_Report.pbix` in Power BI Desktop
3. Refresh data source if connecting to live data
4. Explore the report pages: Summary, Customer Detail, Transaction Analysis
